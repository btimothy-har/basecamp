# engineering

Basecamp engineering policy, tools, and skills — file focus, frontend design, data analysis, BigQuery, Python, SQL, dbt, marimo, and Pi development.

## What it does

- **File-length reminder**: fail-open guidance after a structured edit leaves a recognized source file over its soft cap
- **bq_query tool**: BigQuery SQL execution from `.sql` files under the workspace scratch directory
- **Engineering skills**: code-first frontend design and prototyping, Python development, SQL, data warehousing (dbt), marimo notebooks, data analysis, and Pi development
- **Engineering prompts**: domain-specific prompt templates

## Dependencies

- **core** (`#core/*`): workspace effective cwd, workspace state (scratch dir)
