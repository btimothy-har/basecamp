## Review the Plan

Read today's journal file. Find everything that was planned — the NOW and LATER items set this morning. This is the baseline for what was supposed to happen.

## Discover What Actually Happened

Invoke the `recall` skill to find today's sessions:
1. `recall list --after <today> --cross-project` to find all of today's sessions across projects
2. `recall session <id>` for full context on each session
3. Check GitHub for PRs merged, reviewed, or updated today

Group findings by thread of work, not by session or repo.

## Close the Loop

For each planned item, determine its outcome:
- Completed → mark DONE in the journal
- In progress, continuing tomorrow → leave as NOW or move to LATER
- Dropped or deferred → mark DONE with a note, or remove

Update the journal file directly — don't append duplicates.
